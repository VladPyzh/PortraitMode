1. Person segmentation
To train a person segmentation task the Trainer.ipynb notebook should be executed.
The training dataset can be downloaded here: https://hackernoon.com/releasing-supervisely-person-dataset-for-teaching-machines-to-segment-humans-1f1fc1f28469

2. Depth Map estimation 
To train the depth map estimation model the train-depth-map-estimation-model.ipynb notebook should be executed.
 The training data can be downloaded here: https://drive.google.com/drive/folders/1TzwfNA5JRFTPO-kHMU___kILmOEodoBo

3. Models aggregation
To aggregate the result of two models and to get the final blurring algorithm the inference.ipynb notebook should be executed. 


Our checkpoints are available here: https://drive.google.com/drive/folders/19wQn842DemRZEKT8W4xdrjJGN4M0s5vH?usp=sharing

Input image examples are available in ./images folder.
